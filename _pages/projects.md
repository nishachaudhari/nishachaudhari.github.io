<h2> PassKeep </h2>

**USB Encrypted Password Keeper**

<span style = "color: #00adb5">*August 2019 - May 2020* </span>

This product was proposed to address the growing security risk involving online accounts and passwords, especially regarding those that store personal information about users. 
My colleagues and I created a mock USB dongle on a raspberry pi that generates and stores random encrypted passwords. 
The device uses a website display hosted on the raspberry pi to allow users to manage their passwords.
There is full AES encryption of the users stored data in a directory storage system. 
The software used in this project includes: Flask, Apache, HTML, Python, and Raspian from Raspberry Pi.
Watch our virtual presentation and demo [here](https://www.youtube.com/watch?v=W20VeouqOoU).

<h2>Food Savior</h2>

**Mobile App to help reduce food waste**

<span style = "color: #00adb5">*January 2020 - May 2020* </span>

A cloud based mobile app where users can publish their free available food along with basic information. 
In addition, users can also either explore food in their area in both list and map view or search for specific types of food.They can then reserve the food and pick it up once the food owner accepts their request. 
There is a messaging feature which facilitates communication between users. 
This app was developed using a Flutter framework along with Firebase and Google Cloud Platform APIs.
